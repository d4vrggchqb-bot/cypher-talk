A multi-user platform leveraging modern database architecture and classic cipher techniques for ultimate privacy.

MISSION: MULTI-USER PRIVACY VIA CIPHER

CypherTalks is designed to facilitate secure, multiple-user communication. We serve as the platform where users **encrypt their messages using a chosen cipher before sending** and **decrypt messages upon reception**. The database stores only the encrypted cyphertext.

CypherTalks is Good to Use: This website is excellent for users who want complete control over their security. By relying on client-side encryption (ciphering), we ensure that even if our database were compromised, the data remains unreadable. **We use Supabase for robust authentication and real-time database capabilities.**

Our Core Purpose

  • Control:Empower users to manage their own encryption keys/methods.
  • Platform:Utilizes **Supabase** for secure user management and real-time chat data handling.
  • Data: The platform *only* stores cyphertext, ensuring server-side privacy.
